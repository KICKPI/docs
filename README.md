Technical documentation for KICKPI development boards, including setup, image flashing, SDK compilation, customization, and troubleshooting.

## Community Notice

This is an independent community repository. See [DISCLAIMER.md](DISCLAIMER.md) for non-affiliation and project intent.

## Scope

This repository is the central place for KICKPI documentation.

- Board introductions and hardware notes
- Getting started and bring-up guides
- Image flashing workflows
- Android and Linux SDK build guides
- Driver and system customization notes
- Debugging and troubleshooting procedures

## Repository Goals

- Provide clear, practical, and reproducible instructions
- Keep board-specific and tool-specific docs well organized
- Improve developer onboarding and support efficiency
